# myCombSort

Реализация алгоритма «Сортировка расчёской» (Comb Sort) на C++

[![GitHub Actions](https://github.com/BsMaIL0/myCombSort/actions/workflows/ci.yml/badge.svg)](https://github.com/BsMaIL0/myCombSort/actions/workflows/ci.yml)
[![Tests](https://img.shields.io/badge/tests-19%20passed-brightgreen)](https://github.com/BsMaIL0/myCombSort)

## 📋 Обзор

Сортировка расчёской — это улучшенная версия пузырьковой сортировки, которая устраняет её главный недостаток — медленное перемещение «черепах» (маленьких элементов в конце массива).

В отличие от пузырьковой сортировки, которая сравнивает только соседние элементы (расстояние = 1), Comb Sort начинает с **большого расстояния** между сравниваемыми элементами, постепенно уменьшая его до 1. Это позволяет быстро перемещать элементы на большие расстояния, напоминая расчёсывание волос.

## ✨ Возможности

- **Шаблонная реализация** — работает с любыми типами данных
- **Поддержка компараторов** — сортировка в любом порядке (возрастание, убывание, пользовательский)
- **Сортировка на месте** — без выделения дополнительной памяти

## 🚀 Быстрый старт

### Базовое использование

```cpp
#include "myCombSort.h"

int main() {
    // Сортировка целых чисел
    int numbers[] = {64, 34, 25, 12, 22, 11, 90};
    myCombSort(numbers, 7);
    // Результат: {11, 12, 22, 25, 34, 64, 90}
    
    // Сортировка вещественных чисел
    double decimals[] = {3.14, 1.41, 2.71, 1.0};
    myCombSort(decimals, 4);
    // Результат: {1.0, 1.41, 2.71, 3.14}
    
    // Сортировка строк
    std::string words[] = {"banana", "apple", "cherry", "date"};
    myCombSort(words, 4);
    // Результат: {"apple", "banana", "cherry", "date"}
    
    // Пользовательский компаратор (убывание)
    int arr[] = {1, 2, 3, 4, 5};
    myCombSort(arr, 5, std::greater<int>());
    // Результат: {5, 4, 3, 2, 1}
    
    // Сортировка по абсолютному значению
    int negArr[] = {-5, 2, -1, 4, -3};
    auto absCompare = [](int a, int b) { return std::abs(a) < std::abs(b); };
    myCombSort(negArr, 5, absCompare);
    // Результат: {-1, 2, -3, 4, -5} (по |1|,|2|,|3|,|4|,|5|)
    
    return 0;
}
```

## 📊 Сложность алгоритма

| Случай | Временная сложность | Пространственная сложность |
|--------|---------------------|----------------------------|
| **Лучший** (уже отсортирован) | O(n log n) | O(1) |
| **Средний** | O(n²/2^p) | O(1) |
| **Худший** | O(n²) | O(1) |

**Примечание:** Comb Sort значительно быстрее пузырьковой сортировки в среднем случае благодаря использованию переменного шага. Оптимальный коэффициент `1.247` был найден эмпирически и обеспечивает наилучшее уменьшение шага.

## 🔧 Сборка и тестирование

### Требования

- CMake 3.20 или выше
- Компилятор с поддержкой C++17
- GoogleTest (будет загружен автоматически через `FetchContent`)

### Инструкция по сборке

```bash
# Клонирование репозитория
git clone <url-вашего-репозитория>
cd myCombSort

# Конфигурация CMake
cmake -B build -DCMAKE_BUILD_TYPE=Release

# Сборка проекта
cmake --build build --config Release

# Запуск тестов
cd build
ctest --output-on-failure --verbose

# Запуск тестов вручную

# Запуск всех тестов
./build/tests

# Запуск определённых тестов
./build/tests --gtest_filter=CombSortTest.*
./build/tests --gtest_filter=*LargeArray*
```
